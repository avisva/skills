Getting Started with JavaScript

JavaScript Essentials language

Variables, types, functions

JavaScript is a dynamically typed language that runs in the browser (and on servers using Node.js). Use let and const instead of var for predictable scoping. 

let name = "Alice";
const age = 30;

function greet(person) {
  return `Hello, ${person}!`;
}

const greetArrow = (person) => `Hi, ${person}!`;

Functions are first‑class values—you can assign them to variables, pass them around, and return them from other functions.

DOM & Events browser

Selecting elements and handling user actions

The DOM (Document Object Model) represents your HTML as a tree of objects. Use JavaScript to read and update this tree in response to user actions. 

const button = document.querySelector("button");

button.addEventListener("click", () => {
  alert("Clicked!");
});

Event listeners let you respond to clicks, keypresses, and many other browser events, making your pages interactive.
